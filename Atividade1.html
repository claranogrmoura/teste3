HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade

Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outreas tecnologias
Muitos recursos do HTML têm sido construído com a intenção de serem executados em dispositivos móveis como smartphone e tablets

História

Entre 1991 e 1999, o HTML se desenvolveu a partir da versão 1 até a 4. Em 2008 a primeira versão pública do HTML foi lançada

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a Web. Com o tempo aumentou as tendências em tornar sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web desenvolvidas com as primeiras versões do HTML.
